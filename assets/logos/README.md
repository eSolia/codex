# eSolia Logo Assets

All logos have transparent backgrounds and work on various colored surfaces.

## Logo Types

| Type | Description | Use Case |
|------|-------------|----------|
| **horiz** | Horizontal logo (symbol + name) | Headers, documents, proposals |
| **verti** | Vertical/square logo (symbol + name stacked) | Square spaces, favicons at larger sizes |
| **symbol** | Symbol only with padding | Icons, small spaces with margin |
| **symbolcropped** | Symbol only, no padding | Tight layouts, edge-to-edge |

## Color Variants

| Color | Hex | Best On |
|-------|-----|---------|
| **darkblue** | `#2D2F63` | White/light backgrounds |
| **black** | `#000000` | Light backgrounds |
| **grey** | `#6b7280` | Light backgrounds, subtle use |
| **white** | `#FFFFFF` | Dark/navy backgrounds |
| **mono** | — | Grayscale contexts |
